什么是 SmartDNS Edge？

What is SmartDNS Edge?

SmartDNS Edge 是使用 Rust 语言重写的本地 DNS 服务器。它能够获取多个上游 DNS 的解析结果，并通过测速算法动态获取最快的网站 IP 响应，从而显著提升网络访问相应速度与网页加载体验。

SmartDNS Edge is a local DNS server rewritten in the Rust programming language. It resolves DNS queries from multiple upstream servers, dynamically determines the fastest IP addresses using smart speed-testing algorithms, and significantly improves internet access speeds and web-loading experiences.

✨ 核心特性

✨ Core Features

  • 绿色工具软件：解包即用，需要自己编辑 smartdns.conf 配置文件，可作为服务安装。
  • Green Utility Software: Out-of-the-box, easy configuration with a single smartdns.conf file, and supports installation as a system service.
  • 前沿加密协议支持：全面支持 DNS-over-TLS (DoT)、DNS-over-HTTPS (DoH)、DNS-over-QUIC (DoQ) 以及最新的 DNS-over-HTTP/3 (DoH3)。
  • Cutting-edge Encryption Protocol Support: Full support for DNS-over-TLS (DoT), DNS-over-HTTPS (DoH), DNS-over-QUIC (DoQ), and the latest DNS-over-HTTP/3 (DoH3).
  • 持久缓存（乐观缓存）：完善的新鲜缓存、过期缓存预取、超时缓存、否定缓存机制，后台自动更新满足热度要求的过期缓存，前端解析响应毫秒级响应。
  • Persistent Cache (Optimistic Cache): Comprehensive mechanisms for fresh cache, prefetching expired cache, timeout cache, and negative cache. It automatically updates expired cache in the background based on request frequency, achieving millisecond-level DNS response times.
  • 高性能 SOCKS5/HTTP 代理隧道：内置零分配（Zero-Allocation）的高并发 SOCKS5 代理客户端，安全绕过代理漏流。
  • High-Performance SOCKS5/HTTP Proxy Tunneling: Built-in zero-allocation, high-concurrency SOCKS5 proxy client that securely avoids proxy leaks.
  • 全平台兼容：完美兼容 Windows、Linux、macOS，支持无管理员权限（Unprivileged ICMP）极速 Ping 测速。
  • Cross-Platform Compatibility: Fully compatible with Windows, Linux, and macOS. Supports unprivileged ICMP ping for speed testing without administrator/root privileges.
  • 智能防漏流与分流：支持高级防火墙 SO_MARK、网卡设备绑定以及多 WAN 口策略路由。
  • Smart Routing & Leak Prevention: Supports advanced firewall tagging (SO_MARK), binding sockets to specific network interfaces, and multi-WAN policy routing.
  • 极高并发处理性能：CPU多核心优化，极高网络并发处理能力，无内存泄漏。
  • Ultra-high Concurrency: Optimized for multi-core CPUs with extreme network concurrency handling and zero memory leaks.
  • 完美的双栈优选：A与AAAA记录进行优选，选出最优访问速度的记录，默认参数下，只要有A记录，A记录始终有效，如果AAAA纪录更优，那就都保留；可设置AAAA记录优先，哪个记录更优就只保留哪个记录。A和AAAA记录TTL始终对齐
  • Optimal Dual-Stack Selection: Optimizes IPv4 (A) and IPv6 (AAAA) records to choose the fastest access route. By default, as long as an A record exists, it is kept valid. If the AAAA record is faster, both are kept. You can also prioritize AAAA records so only the faster one is returned. A and AAAA record TTLs are always kept aligned.
  • 漏洞防御与反网络攻击：防御恒定时间防侧信道爆破、源地址投毒清洗、UDP放大反射、HTTPS内存炸弹、慢速握手等网络攻击；使用绝对的流量沙盒防止泄漏真实地址
  • Vulnerability Defense & Anti-Network Attack: Defends against constant-time side-channel attacks, source address poisoning/scrubbing, UDP amplification/reflection, HTTPS memory bombs, and slow-handshake attacks. Uses isolated sandboxes to prevent real IP leaks.
  • resolve命令行工具：支持完善的resolve功能，对标dig
  • Resolve CLI Tool: Built-in feature-complete resolution command-line tool, fully matching the capabilities of dig.

🪟 Windows (企业服务器 & 个人桌面)

🪟 Windows (Enterprise Servers & Desktops)

将下载的 .zip 文件解压到一个固定目录（如 D:\SmartDNS）。

Extract the downloaded .zip file to a fixed directory (e.g., D:\SmartDNS).

方式一：前台测试运行（适合测试排障）

Method 1: Foreground Test Execution (Best for troubleshooting)

.\smartdns.exe run -c .\smartdns.conf -v

(注：-v 表示开启调试日志输出，方便直观查看解析过程)

(Note: -v enables debug log output to visually inspect the resolution process)

方式二：后台服务运行（推荐，开机自启）

Method 2: Background Service Execution (Recommended, Autostart)

请以管理员身份运行终端（PowerShell），执行以下命令，系统将全自动接管进程并配置好防火墙规则：

Run your terminal (PowerShell) as Administrator and execute the following commands. The system will automatically register the background process and configure the local firewall rules:

# 1. 安装服务
.\smartdns.exe service install

# 2. 启动服务
.\smartdns.exe service start

# 3. 随时查看运行状态 (带 🟢/🔴 指示灯)
.\smartdns.exe service status
# 1. Install service
.\smartdns.exe service install

# 2. Start service
.\smartdns.exe service start

# 3. Inspect service status (with 🟢/🔴 indicators)
.\smartdns.exe service status

(To uninstall, simply execute .\smartdns.exe service uninstall)

Extract the archive to your target directory. Open your terminal, grant execution permissions, and run:
